三层交换机VLAN配置学习

命令

  • System-view

    • sy 进入配置视图

  • vlan [1-4094]

    • 创建 vlan
    • vlan 1 为默认vlan
    • vlan标识只能为数字

  • interface G0/0/1

    • 进入1号口进行配置

  • port link-type [access/trunk]

    • access
      • 只链接1个vlan
    • trunk
      • 可配置多个vlan
    • 端口类型配置

  • port defaule vlan [1-4094]

    • 配置该端口的vlan

  • int vlan [1-4094]

    • 进入vlan

  • ip add 192.168.1.254 [255.255.255.0/24]

    • 对该vlan配置网关地址

  • dis ip int b

    • 查看vlan配置
